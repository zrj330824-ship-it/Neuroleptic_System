## 🌀 核心数学框架

### 1. 神经场方程

```python
# 连续神经场动力学
∂u(x,t)/∂t = -u(x,t) + ∫ w(x,y) · f(u(y,t)) dy + I(x,t)

其中:
- u(x,t): 位置 x、时间 t 的神经活动
- w(x,y): 连接核（突触权重分布）
- f(·): 激活函数（sigmoid）
- I(x,t): 外部输入

这不是在"计算"，是在"演化"！
```

### 2. 吸引子动力学

```
初始状态 → 演化轨迹 → 吸引子态

推理 = 从初始条件演化到吸引子
记忆 = 吸引子的位置
学习 = 改变吸引子 landscape
```

### 3. 符号作为吸引子

```
传统符号 AI: 符号是离散 token
NFC: 符号 = 场的稳定模式（吸引子）

例如 "三角形" 不是字符串，而是:
- 场的特定空间振动模式
- 三个激活峰的稳定构型
- 可以连续变形但保持拓扑不变
```

## 💡 如何实现"逻辑推理"

### 传统方式（错误）
```python
if A and B:
    return C
```

### NFC 方式（正确）
```
1. "A" 编码为场的模式 A
2. "B" 编码为场的模式 B
3. "A ∧ B → C" 编码为场的连接结构 w(x,y)
4. 初始化场 = A + B
5. 让场自由演化
6. 场自发重组，稳定到 C 的模式
7. 读取 C

逻辑推理 = 场的自发演化轨迹
```

### 关键洞见
```
逻辑规则不是"执行的代码"
逻辑规则是"场的连接结构"

推理不是"运行算法"
推理是"场沿能量梯度下滑到吸引子"
```

## 🧬 学习机制

### 传统学习（反向传播）
```
预测 → 误差 → 反向传播 → 更新权重
```

### NFC 学习（场结构重塑）
```
1. 暴露于模式
2. 场产生共振
3. 共振导致结构变化（Hebbian 学习）
4. 下次同样模式更容易激活

学习不是"优化损失"
学习是"场的结构因经验而改变"
```

### 数学形式
```
Hebbian 学习规则:
Δw(x,y) = η · u(x) · u(y)

"一起发放的神经元连在一起"
这塑造了场的演化 landscape
```

### 阶段 1: 基础场动力学（1-2 月）

**核心方程实现**:
```python
class NeuralField:
    def __init__(self, shape, kernel):
        self.shape = shape  # 场的空间维度
        self.kernel = kernel  # 连接核 w(x,y)
        
    def evolve(self, initial_state, time_steps):
        """场的时间演化"""
        state = initial_state
        for t in range(time_steps):
            # 神经场方程
            du = -state + convolve(self.kernel, f(state))
            state = state + dt * du
        return state
```

**关键突破点**:
- 连续空间表示（不是离散 token）
- 偏微分方程数值解
- 吸引子检测

### 阶段 3: 推理作为演化（3-4 月）

**核心思想**:
```
前提 A, B 编码为初始场态
逻辑规则编码为 kernel 结构
推理 = 场从 (A,B) 演化到 C
```

**实现**:
```python
class ReasoningField(NeuralField):
    def __init__(self):
        super().__init__()
        
        # 编码逻辑规则为 kernel
        self.kernel = self.encode_logic_rules()
        
    def infer(self, premises):
        """从前提推理结论"""
        # 1. 前提编码为初始态
        initial = self.encode_premises(premises)
        
        # 2. 场演化
        final_state = self.evolve(initial, time_steps=100)
        
        # 3. 读取结论
        conclusion = self.decode(final_state)
        
        return conclusion
    
    def encode_logic_rules(self):
        """
        将逻辑规则编码为连接核
        
        例如 "A ∧ B → C" 编码为:
        - A 区域和 B 区域同时激活时
        - 通过 kernel 耦合，激发 C 区域
        """
        kernel = np.zeros(self.shape)
        
        # A ∧ B → C 的编码
        kernel[A_region, B_region] = 0.5
        kernel[B_region, C_region] = 0.8
        kernel[A_region, C_region] = 0.8
        
        return kernel
```
