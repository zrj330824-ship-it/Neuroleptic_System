## 🎯 Polymarket 价格调节机制

### 1. 订单簿机制

**Polymarket 使用 CLOB (Central Limit Order Book)**:

```
买方订单簿                    卖方订单簿
价格    数量                  价格    数量
0.51    1000                  0.53    800
0.50    2000                  0.54    1200
0.49    3000                  0.55    1500
...
```

**价格形成**:
- 最新成交价 = 最新匹配订单的价格
- 买一价 = 最高买单价格
- 卖一价 = 最低卖单价格
