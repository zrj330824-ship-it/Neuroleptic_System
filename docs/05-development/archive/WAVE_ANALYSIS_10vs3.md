# NeuralFieldNet 波段收益分析

**分析时间**: 2026-02-26 20:16  
**核心问题**: 一次大上升周期，3 次波段 vs 10 次波段，收益差距多大？  
**结论**: 惊人！10 次波段 = 股神级别收益

---

## 📊 场景假设

### 大上升周期特征

```
周期长度：5-10 分钟
总涨幅：10-20%
波动率：中等偏高
```

---

## 🎯 方案对比

### 方案 A: 只抓 3 次大波段

**操作**:
```
T+0s    价格 0.50, 买入 (底仓 3%)

T+60s   上涨 10% → 一级拐点，卖出 3%
        获利：10% × 3% = 0.30%

T+120s  回调后上涨 8% → 二级拐点，卖出 1.5%
        获利：8% × 1.5% = 0.12%

T+180s  再次上涨 5% → 三级拐点，卖出 0.5%
        获利：5% × 0.5% = 0.025%

总收益：0.30% + 0.12% + 0.025% = 0.445%
```

**年化**:
```
0.445% × (24×60/10) × 365 = 0.445% × 144 × 365 = **23,427%**
(理论上，实际不可能这么频繁)

实际假设：每天 2 次大周期
0.445% × 2 × 250 天 = **222.5%/年**
```

---

### 方案 B: 抓 10 次波段

**操作**:
```
大波段 (3 次):
- 10% × 3% = 0.30%
- 8% × 1.5% = 0.12%
- 5% × 0.5% = 0.025%
小计：0.445%

小波段 (7 次):
- 3% × 0.5% = 0.015%
- 2% × 0.5% = 0.01%
- 4% × 0.5% = 0.02%
- 2.5% × 0.5% = 0.0125%
- 3.5% × 0.5% = 0.0175%
- 2% × 0.5% = 0.01%
- 3% × 0.5% = 0.015%
小计：0.10%

总收益：0.445% + 0.10% = 0.545%
```

**年化**:
```
实际假设：每天 2 次大周期
0.545% × 2 × 250 天 = **272.5%/年**
```

**提升**: 272.5% / 222.5% = **1.22 倍** (+22%)

---

### 方案 C: 极端情况 - 10 次全是小波段

**操作**:
```
每次平均涨幅 2.5%, 仓位 0.5%
单次收益：2.5% × 0.5% = 0.0125%

10 次总收益：0.0125% × 10 = 0.125%
```

**年化**:
```
0.125% × 2 × 250 天 = **62.5%/年**
```

**结论**: 只抓小波段收益低，必须抓住大波段！

```

```
