## 📋 概述

本文档定义 NeuralFieldNet 系统的投资政策、资金配置原则、风险偏好、以及长期投资目标。

---

## 🎯 投资目标

### 长期目标 (1 年)

| 目标 | 目标值 | 说明 |
|------|--------|------|
| **年化收益** | 50-100% | 复利增长 |
| **最大回撤** | <-20% | 风险控制 |
| **夏普比率** | >2.0 | 风险调整后收益 |
| **胜率** | >60% | 交易成功率 |

### 中期目标 (1 季度)

| 目标 | 目标值 | 说明 |
|------|--------|------|
| **季度收益** | 15-25% | 稳定增长 |
| **月均收益** | 5-8% | 月度目标 |
| **最大回撤** | <-10% | 季度风控 |

### 短期目标 (1 月)

| 目标 | 目标值 | 说明 |
|------|--------|------|
| **月收益** | 5-10% | 月度目标 |
| **周收益** | 1-3% | 周度目标 |
| **日收益** | 0.1-0.5% | 日常目标 |

## 💰 资金配置

### 策略资金分配

```
总资金：100%
│
├── 流动性驱动策略：50%  (核心策略，稳定收益)
│
├── 双边套利策略：30%   (低风险，稳定现金流)
│
└── 方向性交易策略：20%  (高风险，超额收益)
```

### 动态调整规则

| 市场条件 | 流动性驱动 | 套利 | 方向性 | 说明 |
|---------|-----------|------|--------|------|
| **高波动** | 40% | 20% | 40% | 增加方向性 |
| **低波动** | 60% | 40% | 0% | 减少方向性 |
| **正常** | 50% | 30% | 20% | 标准配置 |
| **危机** | 70% | 30% | 0% | 保守配置 |

### 仓位管理

| 级别 | 单笔仓位 | 总仓位 | 说明 |
|------|---------|--------|------|
| **保守** | 1% | 5% | 低风险偏好 |
| **标准** | 2% | 10% | 中等风险偏好 |
| **激进** | 3% | 15% | 高风险偏好 |

## 🛡️ 风险政策

### 风险限额

| 风险类型 | 限额 | 触发行动 |
|---------|------|---------|
| **日亏损** | -5% | 停止交易 24 小时 |
| **周亏损** | -10% | 停止交易 7 天 |
| **月亏损** | -15% | 重新评估策略 |
| **最大回撤** | -20% | 清盘止损 |

### 风险预算

```
年度风险预算：-20% (最大回撤)
│
├── 季度风险预算：-10%
│   │
│   ├── 月度风险预算：-5%
│   │   │
│   │   ├── 周度风险预算：-2%
│   │   │
│   │   └── 日度风险预算：-1%
```

### 风险分散

| 分散维度 | 规则 | 说明 |
|---------|------|------|
| **市场分散** | 单市场≤5% | 避免集中风险 |
| **策略分散** | 3 策略并行 | 策略对冲 |
| **时间分散** | 分批建仓 | 避免时点风险 |
| **方向分散** | 多空对冲 | 方向中性 |

## 📊 业绩评估

### 评估指标

| 指标 | 计算公式 | 目标值 |
|------|---------|--------|
| **收益率** | (期末 - 期初) / 期初 | >50%/年 |
| **夏普比率** | (收益率 - 无风险利率) / 波动率 | >2.0 |
| **最大回撤** | max(峰值 - 谷值) / 峰值 | <-20% |
| **胜率** | 盈利交易数 / 总交易数 | >60% |
| **盈亏比** | 平均盈利 / 平均亏损 | >1.5 |
| **卡尔玛比率** | 收益率 / 最大回撤 | >2.5 |

### 评估周期

| 周期 | 评估内容 | 评估时间 |
|------|---------|---------|
| **日度** | 盈亏、交易数 | 每日 24:00 |
| **周度** | 周收益、胜率 | 每周日 20:00 |
| **月度** | 月收益、回撤 | 每月 1 日 |
| **季度** | 季度业绩、策略调整 | 每季度首月 5 日 |
| **年度** | 年度业绩、战略调整 | 每年 1 月 15 日 |

### 业绩基准

| 基准 | 目标超越 | 说明 |
|------|---------|------|
| **USDC 存款** | +5%/年 | 无风险收益 |
| **BTC 持有** | +10%/年 | 加密货币基准 |
| **对冲基金** | +20%/年 | 行业基准 |

## 🔄 再投资政策

### 复利策略

```
初始资金：$10,000
│
├── 月收益：5%
│
├── 月复投：100% (全部再投资)
│
└── 预期增长:
    ├── 3 个月：$11,576 (+15.8%)
    ├── 6 个月：$13,401 (+34.0%)
    ├── 12 个月：$17,959 (+79.6%)
    └── 24 个月：$32,254 (+222.5%)
```

### 资金提取规则

| 条件 | 提取比例 | 说明 |
|------|---------|------|
| **月收益>10%** | 提取 20% | 锁定利润 |
| **月收益>20%** | 提取 30% | 大额利润锁定 |
| **季度收益>30%** | 提取 10% | 季度分红 |
| **年度收益>50%** | 提取 20% | 年度分红 |

### 资金追加规则

| 条件 | 追加金额 | 说明 |
|------|---------|------|
| **回撤>-5%** | 不追加 | 正常波动 |
| **回撤>-10%** | 追加 10% | 低位加仓 |
| **回撤>-15%** | 追加 20% | 大幅加仓 |
| **回撤>-20%** | 不追加 | 等待企稳 |

## 📈 市场配置

### 市场选择标准

| 标准 | 阈值 | 说明 |
|------|------|------|
| **24h 成交量** | >$5,000 | 流动性充足 |
| **流动性评分** | >50 | 可交易 |
| **价差** | <5% | 合理价差 |
| **剩余时间** | >24 小时 | 避免即将结算 |

### 市场权重

| 市场类别 | 权重 | 说明 |
|---------|------|------|
| **加密货币** | 30% | 高波动、高收益 |
| **政治选举** | 25% | 趋势明确 |
| **体育赛事** | 20% | 短期机会 |
| **美联储政策** | 15% | 数据驱动 |
| **其他** | 10% | 分散风险 |

### 市场调整

| 触发条件 | 调整动作 | 说明 |
|---------|---------|------|
| **单市场收益>20%** | 减仓 50% | 锁定利润 |
| **单市场连续亏损 5 笔** | 暂停交易 | 重新评估 |
| **市场流动性下降** | 退出市场 | 规避风险 |

## 🚨 紧急政策

### 危机处理

| 危机级别 | 触发条件 | 处理措施 |
|---------|---------|---------|
| **一级** | 日亏损>-5% | 减仓 50% |
| **二级** | 日亏损>-10% | 清仓 80% |
| **三级** | 日亏损>-15% | 全部清仓 |
| **四级** | 系统故障 | 紧急停止 |

### 黑天鹅应对

```
黑天鹅事件发生
│
├── Step 1: 立即停止所有交易
│
├── Step 2: 评估当前持仓风险
│
├── Step 3: 平仓高风险持仓
│
├── Step 4: 保留现金等待机会
│
└── Step 5: 市场企稳后重新评估
```

### 恢复策略

| 阶段 | 仓位 | 说明 |
|------|------|------|
| **阶段 1** | 10% | 试探性建仓 |
| **阶段 2** | 30% | 确认趋势 |
| **阶段 3** | 60% | 正常交易 |
| **阶段 4** | 100% | 完全恢复 |

## 📝 政策审查

### 审查周期

| 审查类型 | 周期 | 负责人 |
|---------|------|--------|
