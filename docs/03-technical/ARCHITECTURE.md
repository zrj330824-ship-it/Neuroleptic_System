## 📦 核心组件

### 1. 数据层 (Data Layer)

**职责**: 获取和处理市场数据

| 组件 | 说明 | 技术 |
|------|------|------|
| **Polymarket API** | 市场数据、订单管理 | REST API |
| **WebSocket Client** | 实时价格更新 | WebSocket |
| **Gamma API** | 市场发现、元数据 | REST API |
| **Data Validator** | 数据验证和清洗 | Python |

**数据流**:
```
Polymarket API → WebSocket → Data Validator → Strategy Layer
```

### 2. 策略层 (Strategy Layer)

**职责**: 生成交易信号

| 策略 | 优先级 | 仓位 | 说明 |
|------|--------|------|------|
| **流动性驱动** | P0 | 50% | 流动性高时进场，低时退出 |
| **双边套利** | P1 | 30% | YES+NO<1.0 时套利 |
| **方向性交易** | P2 | 20% | 神经场预测涨跌 |

**信号生成流程**:
```
市场数据 → 神经场编码 → 能量计算 → 信号生成 → 风险管理
```

### 3. 执行层 (Execution Layer)

**职责**: 执行交易和风险管理

| 模块 | 功能 | 说明 |
|------|------|------|
| **Order Executor** | 订单执行 | 市价单/限价单 |
| **Risk Manager** | 风险控制 | 止盈/止损/仓位 |
| **Position Manager** | 仓位管理 | 持仓跟踪 |
| **Cost Calculator** | 成本计算 | 费用/滑点 |

### 4. 神经场引擎 (Neural Field Engine)

**职责**: 神经场计算和预测

| 模块 | 功能 | 说明 |
|------|------|------|
| **NeuralField** | 神经场动力学 | 能量演化 |
| **AttractorMemory** | 吸引子记忆 | 模式存储 |
| **Perception** | 感知编码 | spaCy NLP |
| **EnergyCalculator** | 能量计算 | 状态评估 |

**核心算法**:
```python
# 神经场能量计算
energy = calculate_field_state(manifold, memory)

# 信号生成
if energy < 0.39:
    signal = 'BUY_YES'
elif energy > 0.85:
    signal = 'BUY_NO'
else:
    signal = 'HOLD'
```

### 5. 监控层 (Monitoring Layer)

**职责**: 系统监控和告警

| 模块 | 功能 | 说明 |
|------|------|------|
| **Log Monitor** | 日志监控 | 错误检测 |
| **Alert System** | 告警系统 | 邮件/短信 |
| **Dashboard** | 可视化界面 | Flask + Chart.js |
| **Metrics Collector** | 指标收集 | 性能统计 |

### 6. 存储层 (Storage Layer)

**职责**: 数据存储和管理

| 存储类型 | 用途 | 技术 |
|---------|------|------|
| **SQLite** | 交易记录 | 结构化数据 |
| **JSON Files** | 配置/状态 | 半结构化 |
| **Log Files** | 日志 | 文本 |
| **Memory** | 缓存 | 内存 |

## 🔄 数据流

### 交易流程

```
1. 数据采集
   Polymarket API → WebSocket → 市场数据
   
2. 信号生成
   市场数据 → 神经场引擎 → 交易信号
   
3. 风险评估
   交易信号 → 风险管理 → 执行决策
   
4. 订单执行
   执行决策 → Order Executor → Polymarket
   
5. 结果记录
   执行结果 → SQLite + 日志
```

### 监控流程

```
1. 日志收集
   各组件 → Log Aggregator → 日志文件
   
2. 指标计算
   交易日志 → Metrics Calculator → 统计指标
   
3. 告警检测
   指标 → Alert Rules → 告警通知
   
4. 可视化
   数据库 → Dashboard → Web 界面
```

## 🔒 安全架构

### 认证与授权

| 组件 | 认证方式 | 说明 |
|------|---------|------|
| **API Key** | HMAC 签名 | Polymarket API |
| **SSH** | 密钥认证 | VPS 访问 |
| **Dashboard** | 本地访问 | localhost only |

### 数据保护

| 数据类型 | 保护方式 | 说明 |
|---------|---------|------|
| **API Key** | .env 文件 | 不提交到 Git |
| **交易记录** | SQLite | 本地存储 |
| **日志** | 日志轮转 | 保留 7 天 |

### 网络安全

| 措施 | 说明 |
|------|------|
| **防火墙** | 仅开放必要端口 |
| **SSH** | 密钥认证 + 禁用密码 |
| **API** | HTTPS 加密通信 |
