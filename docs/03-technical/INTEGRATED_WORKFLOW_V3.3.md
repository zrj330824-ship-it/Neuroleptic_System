```

│     ├─ 交易量 (24h volume)              │
│     ├─ 订单簿深度                       │
│     └─ 价差 (spread)                    │
│                                         │
│  2. NeuralField Bot v2.0                │
│     ├─ 接收市场数据                     │
│     ├─ 神经场动力学计算                 │
│     ├─ 生成流动性评分 (0-100)           │
│     ├─ 生成方向预测 (BUY/SELL)          │
│     └─ 计算置信度 (0-1)                 │
│                                         │
│  3. RSS 新闻源 (10+ 个)                  │
│     ├─ Reuters, Bloomberg, CNBC         │
```

### 阶段 2: 策略信号生成

```
┌─────────────────────────────────────────────────────────┐
│              三大模块并行计算                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【策略 1】流动性驱动 (50%)                              │
│  ├─ 输入：NeuralField 输出                             │
│  │   ├─ liquidity_score: 82.5                          │
│  │   ├─ direction: 1 (BUY)                             │
│  │   └─ confidence: 0.87                               │
│  │                                                     │
│  ├─ 判断：liquidity_score >= 75?                       │
│  │   ├─ YES → 生成流动性信号 ✅                        │
│  │   └─ NO → 无信号 ⏸️                                 │
│  │                                                     │
│  └─ 输出：                                              │
│      {type: 'liquidity', direction: 'BUY',             │
│       confidence: 0.91, liquidity_score: 82.5}         │
│                                                         │
│                                                         │
│  【策略 2】双边套利 (30%)                                │
│  ├─ 输入：市场价格                                     │
│  │   ├─ yes_price: 0.48                                │
│  │   └─ no_price: 0.53                                 │
│  │                                                     │
│  ├─ 计算：sum_price = 0.48 + 0.53 = 1.01              │
│  │   ├─ sum_price < 0.9975? → NO → 无套利 ⏸️          │
│  │   └─ sum_price >= 0.9975? → 无套利机会             │
│  │                                                     │
│  └─ 输出：None (本次无套利机会)                        │
│                                                         │
│                                                         │
│  【策略 3】方向性交易 (20%)                              │
│  ├─ 输入：NeuralField 输出 + 动量因子                  │
│  │   ├─ confidence: 0.87                               │
│  │   ├─ direction: 1                                   │
│  │   └─ momentum: 0.18                                 │
│  │                                                     │
│  ├─ 判断：confidence >= 0.80 AND |momentum| >= 0.15?  │
│  │   ├─ YES → 生成方向性信号 ✅                        │
│  │   └─ NO → 无信号 ⏸️                                 │
│  │                                                     │
│  └─ 输出：                                              │
│      {type: 'directional', direction: 'BUY',           │
│       confidence: 0.83, momentum: 0.18}                │
│                                                         │
│                                                         │
│  【增强】Alpha Momentum (NLP 新闻)                       │
│  ├─ 输入：50 条新闻                                     │
│  │   ├─ spaCy 情绪分析                                 │
│  │   ├─ 事件检测 (finance/tech/politics)              │
│  │   └─ 实体提取                                       │
│  │                                                     │
│  ├─ 计算：动量信号                                     │
│  │   ├─ sentiment_score: 0.65 (positive)              │
│  │   ├─ events: [fed, interest rate]                  │
│  │   └─ momentum_strength: 0.78                        │
│  │                                                     │
│  └─ 输出：                                              │
│      {type: 'news_momentum', direction: 'BUY',         │
│       confidence: 0.85, momentum_strength: 0.78,       │
│       news_boost: +0.10}                               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 阶段 3: 信号整合

```
┌─────────────────────────────────────────┐
│         信号整合器 (Signal Mixer)        │
├─────────────────────────────────────────┤
│                                         │
│  输入信号：                             │
│  1. ✅ 流动性信号 (priority: 2)         │
│     confidence: 0.91, weight: 50%       │
│                                         │
│  2. ⏸️ 套利信号 (无)                    │
│                                         │
│  3. ✅ 方向性信号 (priority: 3)         │
│     confidence: 0.83, weight: 20%       │
│                                         │
│  4. ✅ NLP 新闻增强 (priority: 1)        │
│     momentum_strength: 0.78             │
│                                         │
│  整合逻辑：                             │
│  ┌─────────────────────────────────┐   │
│  │ 1. 应用 NLP 新闻增强              │   │
│  │    IF momentum_strength >= 0.7  │   │
│  │    THEN confidence += 0.10      │   │
│  │                                 │   │
│  │ 2. 加权计算综合置信度            │   │
│  │    final_confidence =           │   │
│  │      (0.91 * 0.50) +            │   │
│  │      (0.83 * 0.20) +            │   │
│  │      (0.85 * 0.30) = 0.88       │   │
│  │                                 │   │
│  │ 3. 确定最终方向                 │   │
│  │    所有信号都是 BUY → BUY       │   │
│  └─────────────────────────────────┘   │
│                                         │
│  输出：                                 │
│  {                                      │
│    direction: 'BUY',                    │
│    confidence: 0.88,                    │
│    strategies: ['liquidity',            │
│                 'directional'],         │
│    news_boosted: true                   │
│  }                                      │
└─────────────────────────────────────────┘
```

## 🎯 关键决策点

### 决策点 1: 流动性判断

```python
if liquidity_score >= 75:
    # 高流动性，可以交易
    generate_liquidity_signal()
else:
    # 流动性不足，观望
    skip_liquidity_trade()
```

### 决策点 2: 套利机会

```python
if yes_price + no_price < 0.9975:
    # 存在套利空间 (>0.25%)
    execute_arbitrage()
else:
    # 无套利机会
    skip_arbitrage()
```

### 决策点 3: 方向性交易

```python
if confidence >= 0.80 and abs(momentum) >= 0.15:
    # 高置信度 + 强动量
    generate_directional_signal()
else:
    # 条件不足
    skip_directional()
```

### 决策点 4: NLP 增强

```python
if news_momentum_strength >= 0.7:
    # 强新闻动量，增强置信度
    final_confidence += 0.10
    news_boosted = True
```

### 决策点 5: 风险控制

```python
if (confidence >= 0.75 and 
    positions < 5 and 
    capital >= 1000):
    # 通过所有检查
    execute_trade()
else:
    # 风控未通过
    reject_trade()
```
