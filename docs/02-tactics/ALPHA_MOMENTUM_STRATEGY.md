## 📋 概述

阿尔法动量策略基于"利润率优先"原则，扫描高利润率 + 高交易量市场，在阿尔法时刻全力出击。

---

## 🎯 核心逻辑

### 利润率优先原则

**首要衡量标准**: 
```
预期利润率 = (预期收益 - 交易成本) / 投入资金
```

### 交易量辅助

**次要指标**: 24 小时交易量确保"量大管饱"

### 阿尔法时刻

**最佳机会**: 
```
高利润率 (>5%) + 高交易量 (>10 万 USDC) = 最大仓位全力出击
```

## 💰 仓位管理

| 信号强度 | 利润率 | 交易量 | 仓位比例 |
|---------|--------|--------|---------|
| **阿尔法时刻** | >5% | >10 万 USDC | 100% 最大仓位 |
| **高机会** | 3-5% | >5 万 USDC | 50-75% 仓位 |
| **中等机会** | 1-3% | >1 万 USDC | 25-50% 仓位 |
| **低机会** | <1% | 任意 | 观望 |

---

## 📈 交易成本模型

```
有效成本 = Taker 费率 - Maker 返还

净利润 = 预期利润 - 对冲成本 - 滑点 - 有效成本

执行条件:
- 净利润 > 0
- 利润率 >= 0.5%
```
