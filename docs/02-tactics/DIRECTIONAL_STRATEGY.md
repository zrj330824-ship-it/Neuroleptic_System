## 📋 概述

方向性交易策略利用 NeuralField 神经场系统的预测能力，判断市场价格走势方向，做多 (BUY YES) 或做空 (BUY NO) 以获取方向性收益。

---

## 🎯 策略原理

### 神经场预测

```
市场信息 → 神经场编码 → 能量状态 → 方向判断
   ↓           ↓           ↓          ↓
价格/成交量  文本嵌入    吸引子状态  BUY_YES/BUY_NO
```

### 能量阈值

| 能量范围 | 信号 | 置信度 | 操作 |
|---------|------|--------|------|
| **E < 0.39** | BUY_YES | 90-95% | 强烈做多 |
| **0.39 < E < 0.60** | BUY_YES | 60-70% | 谨慎做多 |
| **0.60 < E < 0.85** | HOLD | 30-50% | 观望 |
| **E > 0.85** | BUY_NO | 80-95% | 强烈做空 |

---

## 📊 信号生成

### 信号流程

```
1. 数据采集
   - 市场价格
   - 成交量
   - 订单簿
   
2. 神经场处理
   - 感知编码 (spaCy)
   - 场演化 (50 步)
   - 能量计算
   
3. 信号判断
   - 能量 < 0.39 → BUY_YES
   - 能量 > 0.85 → BUY_NO
   - 其他 → HOLD
   
4. 风险管理
   - 仓位计算
   - 止盈/止损设置
```

### 信号强度

| 强度 | 能量范围 | 仓位 | 说明 |
|------|---------|------|------|
| **强多** | E < 0.30 | 3% | 极高置信度 |
| **中多** | 0.30 < E < 0.39 | 2% | 高置信度 |
| **弱多** | 0.39 < E < 0.60 | 1% | 中等置信度 |
| **观望** | 0.60 < E < 0.85 | 0% | 不确定 |
| **弱空** | 0.85 < E < 0.90 | 1% | 中等置信度 |
| **中空** | 0.90 < E < 0.95 | 2% | 高置信度 |
| **强空** | E > 0.95 | 3% | 极高置信度 |

## 💰 仓位管理

### 仓位计算

```python
# 基础仓位
base_position = 0.02  # 2%

# 根据置信度调整
confidence = calculate_confidence(energy)
position = base_position * confidence

# 示例
if energy = 0.25:  # 强多
    confidence = 0.95
    position = 0.02 * 0.95 = 1.9%
    
if energy = 0.50:  # 弱多
    confidence = 0.65
    position = 0.02 * 0.65 = 1.3%
```

### 仓位限制

| 限制类型 | 值 | 说明 |
|---------|-----|------|
| **单笔最大** | 3% | 强信号 |
| **单市场最大** | 5% | 避免集中 |
| **总仓位最大** | 10% | 总体风险 |

## 🎯 止盈/止损

### 固定止盈/止损

| 类型 | 值 | 说明 |
|------|-----|------|
| **止盈** | +3% | 快速获利 |
| **止损** | -2% | 快速止损 |

### 追踪止损

```python
# 追踪止损配置
trailing_stop = {
    'enabled': True,
    'activation_threshold': 0.02,  # 盈利 2% 后激活
    'trailing_distance': 0.01,     # 追踪距离 1%
}

# 示例
entry_price = 0.50
price_rises_to = 0.52  # +4%
# 追踪止损激活
stop_loss = 0.52 - 0.01 = 0.51  # 锁定 +2% 利润
```

### 时间退出

| 条件 | 操作 | 说明 |
|------|------|------|
| **持仓>12 小时** | 强制平仓 | 避免过夜风险 |
| **23:00 后** | 强制平仓 | 日终清仓 |
| **流动性下降** | 立即平仓 | 规避风险 |

## 📈 策略配置

### 核心参数

```python
DIRECTIONAL_CONFIG = {
    # 能量阈值
    'thresholds': {
        'strong_buy': 0.30,    # 强多
        'buy': 0.39,           # 做多
        'hold_upper': 0.85,    # 观望上限
        'sell': 0.85,          # 做空
        'strong_sell': 0.95,   # 强空
    },
    
    # 仓位配置
    'position': {
        'base': 0.02,          # 基础仓位 2%
        'max_single': 0.03,    # 单笔最大 3%
        'max_market': 0.05,    # 单市场最大 5%
    },
    
    # 退出配置
    'exit': {
        'take_profit': 0.03,   # +3% 止盈
        'stop_loss': 0.02,     # -2% 止损
        'trailing_stop': True, # 启用追踪止损
        'max_hold_hours': 12,  # 最长持仓 12 小时
    },
    
    # 神经场配置
    'neural_field': {
        'attractors': 20,      # 吸引子数量
        'think_steps': 30,     # 思考步数
        'learning_rate': 0.1,  # 学习率
    }
}
```

## ⚠️ 风险控制

### 风险类型

| 风险 | 描述 | 控制措施 |

| **方向风险** | 预测错误 | 止损 -2% |

## 🔄 策略优化

### 参数优化

```python
# 优化方向
optimization_targets = {
    'energy_thresholds': [0.35, 0.40, 0.45],  # 能量阈值
    'take_profit': [0.02, 0.03, 0.04],        # 止盈
    'stop_loss': [0.015, 0.02, 0.025],        # 止损
    'position_size': [0.015, 0.02, 0.025],    # 仓位
}
```

### 自适应调整

```python
# 根据市场波动调整
if market_volatility > 0.5:  # 高波动
    position_size *= 0.7      # 减少仓位
    stop_loss = 0.03          # 放宽止损
elif market_volatility < 0.2:  # 低波动
    position_size *= 1.3      # 增加仓位
    stop_loss = 0.015         # 收紧止损
```
