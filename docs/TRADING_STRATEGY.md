## 策略总览

### 核心理念

> **流动性驱动 + 双向交易 + 神经场预测**

| 特性 | 说明 |
|------|------|
| **市场** | Polymarket (预测市场) |
| **交易时间** | 7×24 小时 |
| **交易对** | YES/NO 份额 |
| **策略类型** | 流动性驱动 + 方向性 + 套利 |
| **持仓时间** | 流动性决定 (通常 2-8 小时) |
| **过夜风险** | 无 (流动性下降自动退出) |

### 三大策略组合

```
┌─────────────────────────────────────────────────────────────┐
│                    NeuralFieldNet 策略系统                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  策略 A: 流动性驱动 (核心)                                   │
│  - 检测市场流动性 → 高时进场，低时退出                       │
│  - 优势：避免流动性枯竭风险，双向获利                        │
│                                                             │
│  策略 B: 双边套利 (稳定收益)                                 │
│  - YES+NO < 1.0 → 同时买入 → 价差回归获利                    │
│  - 优势：低风险，稳定现金流                                  │
│                                                             │
│  策略 C: 方向性交易 (超额收益)                               │
│  - 神经场预测涨跌 → 买入 YES 或 NO                            │
│  - 优势：高收益，可做空                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 策略优先级

| 优先级 | 策略 | 仓位分配 | 预期收益 |
|--------|------|---------|---------|
| **P0** | 流动性驱动 | 50% | +15-25%/月 |
| **P1** | 双边套利 | 30% | +5-10%/月 |
| **P2** | 方向性交易 | 20% | +10-20%/月 |

## 流动性驱动策略

### 核心逻辑

```
市场流动性检测
      ↓
评分 ≥ 75 (HIGH) → 开仓交易 ✅
评分 50-74 (MEDIUM) → 谨慎交易 ⚠️
评分 25-49 (LOW) → 平仓退出 🔴
评分 < 25 (NONE) → 禁止交易 ❌
```

### 流动性指标

| 指标 | 进场阈值 | 退出阈值 | 权重 |
|------|---------|---------|------|
| **24h 成交量** | ≥$5,000 | <$1,000 | 25% |
| **买卖价差** | ≤3% | >8% | 25% |
| **挂单深度** | ≥$10,000 | <$2,000 | 25% |
| **交易频率** | ≥10 笔/小时 | <2 笔/小时 | 25% |

### 流动性评分计算

```python
score = (
    25 × (volume_24h / 5000) +      # 成交量得分
    25 × (0.03 / spread) +          # 价差得分
    25 × (depth / 10000) +          # 深度得分
    25 × (trades_per_hour / 10)     # 频率得分
)
```

### 市场分类

| 市场 | 流动性评分 | 状态 | 操作 |
|------|-----------|------|------|
| crypto-sports | 85/100 | HIGH | ✅ 可开仓 |
| politics-election | 92/100 | HIGH | ✅ 可开仓 |
| finance-fed | 78/100 | HIGH | ✅ 可开仓 |
| tech-ai | 55/100 | MEDIUM | ⚠️ 谨慎 |
| climate-carbon | 28/100 | LOW | 🔴 退出 |

### 双向交易优势

**市场疯狂下跌时**:
```
价格暴跌 → 流动性激增 (交易活跃)
   ↓
流动性评分上升 → 触发开仓信号
   ↓
买入 NO 份额 (做空)
   ↓
价格继续下跌 → NO 份额升值
   ↓
获利平仓 → 大赚 ✅
```

**市场疯狂上涨时**:
```
价格暴涨 → 流动性激增
   ↓
买入 YES 份额 (做多)
   ↓
价格继续上涨 → YES 份额升值
   ↓
获利平仓 → 大赚 ✅
```

## 双边套利策略

### 套利原理

```
Polymarket 机制: YES + NO = $1.00 (理论值)

实际情况:
- 流动性不足时：YES + NO < 1.00 (套利机会)
- 流动性恢复时：YES + NO → 1.00 (获利平仓)
```

### 套利条件

| 参数 | 阈值 | 说明 |
|------|------|------|
| **最小价差** | YES+NO < 0.98 | 2% 套利空间 |
| **目标利润** | 1% | 达到即平仓 |
| **最大持仓** | 1% | 单笔仓位 |
| **最长持仓** | 6 小时 | 超时平仓 |

### 套利流程

```
1. 检测到 YES=$0.48, NO=$0.48 (总和$0.96)
   ↓
2. 同时买入:
   - 100 YES @ $0.48 = $48
   - 100 NO @ $0.48 = $48
   - 总成本：$96
   ↓
3. 等待价差回归:
   - YES=$0.50, NO=$0.50 (总和$1.00)
   ↓
4. 卖出平仓:
   - 卖出 100 YES @ $0.50 = $50
   - 卖出 100 NO @ $0.50 = $50
   - 总收入：$100
   ↓
5. 利润：$100 - $96 = $4 (4.2%)
```

## 方向性交易策略

### 神经场预测

```
市场数据 → 神经场编码 → 能量计算 → 交易信号
   ↓           ↓            ↓          ↓
价格/成交量  文本嵌入     吸引子状态  BUY_YES/BUY_NO
```

### 信号生成

| 能量值 | 信号 | 置信度 | 操作 |
|--------|------|--------|------|
| E < 0.39 | BUY_YES | 90-95% | 做多 |
| 0.39 < E < 0.60 | BUY_YES | 65% | 谨慎做多 |
| 0.60 < E < 0.85 | HOLD | 30% | 观望 |
| E > 0.85 | BUY_NO | 80-95% | 做空 |

### 止盈/止损

| 参数 | 值 | 说明 |
|------|-----|------|
| **止盈** | +3% | 快速获利 |
| **止损** | -2% | 快速止损 |
| **追踪止损** | 启用 | 保护利润 |

## 风险管理

### 仓位管理

| 策略 | 单笔仓位 | 最大总仓位 |
|------|---------|-----------|
| 流动性驱动 | 2% | 10% |
| 双边套利 | 1% | 5% |
| 方向性交易 | 2% | 8% |

### 风险控制

| 风险类型 | 控制措施 |
|---------|---------|
| **流动性风险** | 流动性下降自动退出 |
| **方向风险** | 止盈/止损 + 做空对冲 |
| **市场风险** | 分散 5 个市场 |
| **技术风险** | 自动监控 + 日志告警 |

### 最大回撤控制

- **日最大亏损**: -5% → 停止交易 24 小时
- **周最大亏损**: -10% → 停止交易 7 天
- **月最大亏损**: -15% → 重新评估策略

## API 文档

### Polymarket API

**基础 URL**: `https://api.polymarket.com`

| 端点 | 方法 | 说明 |
|------|------|------|
| `/markets` | GET | 获取市场列表 |
| `/markets/{id}/price` | GET | 获取价格 |
| `/markets/{id}/liquidity` | GET | 获取流动性 |
| `/orders` | POST | 下单 |
| `/orders/{id}` | DELETE | 取消订单 |
| `/positions` | GET | 获取持仓 |

### 神经场 API

**模块**: `neural_field_optimized.py`

```python
from neural_field_optimized import NeuralFieldSystem

# 初始化
brain = NeuralFieldSystem(size=64, spacy_model="en_core_web_sm")

# 感知市场
brain.perceive("crypto-sports: bullish high_volume")

# 思考
brain.think(steps=30)

# 获取能量
energy = brain.get_energy()

# 生成信号
if energy < 0.39:
    action = 'BUY_YES'
elif energy > 0.85:
    action = 'BUY_NO'
else:
    action = 'HOLD'
```
